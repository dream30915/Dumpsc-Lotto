## [1.0.0] - 2026-03-12

### 🎉 Initial Release

#### ✨ Added

##### Design & UI
- **Luxury Design System**
  - Premium gradient backgrounds
  - Glass morphism effects
  - Smooth animations and transitions
  - Glow effects on prominent numbers
  - Professional typography (Prompt font)
  - Gold accent color scheme (#fbbf24)

##### Lottery Types
- **80+ Lottery Types** organized into categories:
  - รายวัน (Daily) - 25 types
  - รัฐบาลไทย (Thai Government) - 3 types
  - ฮานอย (Hanoi) - 5 types
  - ต่างประเทศ (International) - 1 type
  - หุ้น (Stock) - 16 types
  - หุ้น VIP (Stock VIP) - 14 types
  - พิเศษ (Special) - 4 types
  - อื่นๆ (Others) - 2 types

##### Features
- **Lottery Selection Page**
  - Beautiful card-based layout
  - Category headers with decorative lines
  - Country flags for each lottery type
  - Current date display
  - Hover effects and animations
  - Responsive grid layout

- **Poster Editor**
  - Real-time preview
  - Customizable lottery name
  - Format selection (VIP, ธรรมดา, พิเศษ)
  - Date picker with random and today buttons
  - Seed input with random generator
  - Single prominent number (เด่น)
  - Three 2-digit numbers
  - Three 3-digit numbers
  - Random all numbers button

- **UI Customization**
  - Adjustable poster size (300-600px width, 500-900px height)
  - Adjustable prominent number size (6-14 rem)
  - Adjustable 2-digit number size (1.5-4 rem)
  - Adjustable 3-digit number size (1.5-4 rem)
  - Slider controls for easy adjustment

- **Background Customization**
  - Default gradient background
  - Upload custom background image
  - Support for all image formats
  - Background size and position optimization

- **Social Media Integration**
  - Facebook name input with branded badge
  - LINE ID input with branded badge
  - Social icons with brand colors

- **Watermark System**
  - DUMPSC.COM watermark
  - Toggle on/off
  - Diagonal pattern overlay
  - Semi-transparent design

- **DUMPSC Branding**
  - Prominent DUMPSC.COM badge
  - Gold gradient background
  - Top-right corner placement
  - Always visible

- **Export & Share**
  - Download as PNG (high quality, 2x scale)
  - Auto-generated filename
  - Share functionality (opens in new window)
  - Preview modal with format selection

##### Responsive Design
- **Mobile Support** (< 600px)

- **Tablet Support** (600px - 900px)
