# การติดตั้งบน Subdomain (lotto.dumpsc.com)

วิธีนี้จะช่วยแยกโปรเจกต์หวยออกจากเว็บหลัก (OpenClaw) อย่างสมบูรณ์ ทำให้ไม่เกิดปัญหา Port ชนกัน และจัดการไฟล์ได้ง่าย

## 1. สร้าง Subdomain ใน Hostinger
1. เข้าสู่ระบบ **Hostinger Control Panel**
2. เลือกจัดการเว็บไซต์ **dumpsc.com**
3. ค้นหาเมนู **Subdomains** (ซับโดเมน) ในแถบด้านซ้าย
4. ในช่อง "Create a New Subdomain" พิมพ์ชื่อ: `lotto`
5. ติ๊กถูกช่อง "Custom folder for subdomain" (ถ้ามี) เพื่อให้แน่ใจว่ามันแยกโฟลเดอร์ชัดเจน
6. กดปุ่ม **Create**

## 2. เตรียมไฟล์
ในเครื่องของคุณ (โปรเจกต์นี้):
1. ตรวจสอบว่ามีโฟลเดอร์ `dist` ที่เพิ่ง Build เสร็จ
2. ตรวจสอบว่ามีไฟล์ `.htaccess` อยู่ในโฟลเดอร์ `dist` แล้ว (ถ้ายังไม่มี ให้ copy จาก `.htaccess.example` ไปใส่)

## 3. อัพโหลดไฟล์
1. เข้าเมนู **File Manager** ใน Hostinger
2. มองหาโฟลเดอร์สำหรับ Subdomain ใหม่
   - ปกติจะอยู่ที่ `public_html/lotto`
   - หรือบางครั้งอาจอยู่ที่ `domains/lotto.dumpsc.com/public_html`
3. เข้าไปในโฟลเดอร์นั้น แล้ว **ลบไฟล์ default.php** หรือ index เดิมที่ Hostinger สร้างไว้ออก
4. **อัพโหลดไฟล์ทั้งหมด** จากโฟลเดอร์ `dist` ในเครื่องคุณ ขึ้นไปไว้ที่นี่
   - ไฟล์ที่ต้องมี: `index.html`, `assets` (โฟลเดอร์), `.htaccess`

## 4. เสร็จสิ้น
เข้าใช้งานได้ทันทีที่: **http://lotto.dumpsc.com**
(หากขึ้นหน้า Error ให้รอประมาณ 15-30 นาทีเพื่อให้ DNS อัพเดท หรือลองกด Ctrl+F5 เพื่อเคลียร์ Cache)
